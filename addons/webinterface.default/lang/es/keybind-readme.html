Enlaces clave

Puede utilizar el teclado para controlar Kodi, esta es la configuración predeterminada, pero se puede cambiar lo que los controles de teclado en la [Página de configuración] (#settings/Web). Las opciones disponibles son:

Kodi

El teclado controla Kodi y la interacción del teclado por defecto con el navegador está desactivado. ( por ejemplo, usando flechas arriba y abajo para desplazarse por una página). [Comando navegador] = [acción en Kodi]. 

Cursor IZQUIERDA = Direccion IZQUIERDA
Cursor DERECHA = Direccion DERECHA
Cursor ARRIBA = Direccion ARRIBA
Cursor ABAJO = Direccion ABAJO
RETROCESO = Atras
INTRO = Seleccionar
TABULADOR = Cerrar
BARRA ESPACIADORA = Reproducir/Pausa
Tecla "C" = Menú contextual 
Tecla "+" = Volumen Arriba
Tecla "-" = Volumen Abajo
Tecla "X" = Parar
Tecla "T" = Alternar subtítulos 
Tecla ">" = Reproducir Siguiente
KTeclaey "<" = Reproducir Anterior
Tecla "\" = Pantalla Completa

Navegador

El teclado sólo controla el navegador. Cuando el remoto está abierto las teclas sólo controlarán Kodi.

Ambos

Comandos de teclado se ejecutan en Kodi y en el navegador. Cuando el control remoto está abierto, las teclas controlan sólo Kodi. Nota: muchos comandos son compartidos por el navegador y Kodi. Eg. Retroceso.
